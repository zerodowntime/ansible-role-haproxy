# haproxy

Ansible role to install and configure HA-Proxy, HA-Proxy is a TCP/HTTP reverse proxy.

## Requirements

This role requires Ansible 2.5 or higher.

Supported platforms:

```yaml
  platforms:
    - name: EL
      versions:
        - 7
```
